# AmongUsIRL
A really bad version of among us but using phones and qr codes. :P


![boi](screenshot.jpg?raw=true "boo")

## Installation
```
$ git clone https://github.com/DroopCat/AmongUsIRL.git
$ cd AmongUsIRL
$ npm install
$ npm start
```
